# 🤖 Agentic AI Course Assistant

An intelligent AI assistant built using **LangGraph, RAG, and real-time web search**.

---

## 🚀 Live Demo
👉 https://agentic-ai-course-assistant-ay78ckshpzodk3j722f8pk.streamlit.app/

---

## 🔑 Features
- Agentic AI using LangGraph
- RAG (Retrieval-Augmented Generation) with ChromaDB
- Real-time web search using Tavily
- Tool-based reasoning
- Deployed on Streamlit Cloud

---

## 🧠 How it works
This system dynamically chooses between:
- 📚 Internal knowledge (ChromaDB)
- 🌐 External web search (Tavily)

to generate accurate responses.

---

## 🛠 Tech Stack
- Python
- LangGraph
- LangChain
- ChromaDB
- Tavily API
- Streamlit
